# Ledger dApp

Welcome to my Week 2 project! This is a "Ledger" application where you can track deposits and withdrawals on the blockchain.

**Live Demo:** [https://pranav-ledger.vercel.app](https://pranav-ledger.vercel.app)

## Features
- **Connect Wallet:** Link your MetaMask account.
- **Deposit ETH:** Send ETH to the smart contract.
- **Withdraw ETH:** Get your ETH back from the contract.
- **Check Balance:** See your current balance in the contract.

## Tech Stack
- HTML, CSS, JavaScript
- viem (for blockchain magic)
- Solidity (smart contracts)
